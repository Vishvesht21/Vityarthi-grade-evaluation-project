# Vityarthi-grade-evaluation-project
Features
•	 Calculates results based on subject marks and overall percentage
•	 Handles single or multiple students
•	 Shows detailed result cards with subject-wise performance
•	 Provides class summary with statistics
•	 Automatic supplementary exam detection
SUBJECT PASSING CRITERIA
•	Minimum 40 marks required in each subject
•	All subjects must be passed individually
•	Subjects: Calculus, Physics, Programming, English, Electronics
 OVERALL PASSING CRITERIA
•	Minimum 40% aggregate percentage required
•	Total marks calculated out of 500 (5 subjects × 100 marks each)
•	Percentage = (Total Marks / 500) × 100
 SPECIAL PROVISION RULE
•	Condition 1: Student fails in EXACTLY ONE subject only
•	Condition 2: Student's overall percentage is 40% or higher
•	Result: PASS but supplementary exam required for the failed subject
•	Note: This provision does NOT apply if failed in more than one subject
 RESULT CATEGORIES
 FULL PASS
•	Passed all individual subjects (≥40 marks each)
•	Achieved ≥40% overall percentage
•	No supplementary exams required
 PASS WITH SUPPLEMENTARY
•	Failed in exactly one subject
•	Overall percentage ≥40%
•	Must appear for supplementary exam in the failed subject
 FAIL
•	Failed in multiple subjects (2 or more), OR
•	Overall percentage below 40% (even if passing all subjects)
•	Must reappear for all failed subjects in next attempt
