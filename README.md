# Vityarthi-grade-evaluation-project
KEY POINTS
Subject passing is mandatory - Each subject must have ≥40 marks

Overall percentage is crucial - Must maintain ≥40% aggregate

One-subject failure exception - Only works if other conditions are met

Multiple failures - Automatic FAIL regardless of overall percentage

Low percentage - FAIL even if passing all subjects but overall <40%

FINAL RESULT DEFINATIONS
Clear Pass: All subjects cleared with ≥40% overall

Pass with Supplementary: Overall passed but one subject supplementary needed

Failed: Multiple subject failures or low overall percentage
